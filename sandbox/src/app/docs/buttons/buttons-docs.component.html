UI Kit предлагает использовать стадартные компоненты кнопок Angular Material, представляя возможности по их стилизации и модификации с помощью cобственно стилевой схемы. Вся подробная информация с описанием базовых компнонентов и доступным API представленна в официальной документации.

Тематизация

Окраска кнопок в соответствии с использемой цветовой темой производится сдандартым путем, с помощью атрибута color. Возможны три стадартных варианта цветовой темой: primary, accent и warn.

Изменение размера

Кнопки могут иметь одит из трех предопределенных рамеров: маленький, средний (базовый размер из Angular Material) и большой. Для изменения размера кнопки используются используются соответствующие CSS модификаторы, которые необходимо указать в атрибуте class компонента кнопки: __size-s, __size-m (полностью идентичен базомовому значению), __size-l.
